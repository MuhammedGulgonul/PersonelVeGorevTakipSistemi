# Personel ve Görev Takip Sistemi

Bu proje, şirketlerde personel, departman ve görev süreçlerinin yönetilmesini sağlayan web tabanlı bir sistemdir.

## Proje Kapsamı ve Modülleri
- **Kullanıcı Giriş Sistemi:** Güvenli login/logout ve şifre değiştirme özellikleri.
- **Personel Yönetimi:** Personel ekleme, güncelleme, silme (aktif/pasif yapma) ve listeleme (CRUD).
- **Departman Yönetimi:** Departman ekleme, güncelleme ve silme.
- **Görev Yönetimi:** Görev oluşturma, personele atama, filtreleme, arama ve durum güncelleme.
- **Görev Dosya Ekleme:** Görevlere PDF veya resim (PNG/JPG) yükleyebilme.
- **Dashboard:** Şirket istatistiklerinin görselleştirilmesi.
- **Raporlama:** Personel listesini Excel'e aktarma ve görev raporlarını PDF olarak indirme.
- **Sistem Logları:** Oluşan hataların arka planda veritabanına kaydedilmesi.

## Kullanılan Teknolojiler
- **Backend:** C# / .NET 8.0 ASP.NET Core MVC
- **Veritabanı & ORM:** MS SQL Server & Entity Framework Core 
- **Frontend:** HTML, CSS, Bootstrap, JavaScript & jQuery
